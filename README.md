# Tennis Match Prediction ML

Machine learning pipeline for ATP tennis match prediction using:
- retrospective Elo ratings,
- rolling player statistics,
- bookmaker market probabilities,
- and time-aware validation.

The project focuses primarily on:
- structured feature engineering,
- leakage prevention,
- temporal evaluation methodology,
- and reproducible analytical workflows.

Raw ATP/Flashscore.com datasets are generated by a separate web-scraping pipeline:

- [tennis-web-scraper](https://github.com/Pianist16/tennis-web-scraper)

---

# Feature Experiment Summary

Primary evaluation methodology:
- Train: 2018–2023
- Validation: 2024
- Test: 2025

Temporal validation was used to avoid future leakage across eras.

## Feature Progression Results

| Feature Set | Logistic Regression Test Accuracy | ROC AUC | Observation |
|---|---:|---:|---|
| Recent Form Only | ~58.6% | ~0.62 | Weak standalone signal |
| Rolling Stats Only | ~58.9% | ~0.63 | Moderate standalone signal |
| Elo Only | ~63.3% | ~0.69 | Strong non-market baseline |
| Market Probabilities Only | ~67.0% | ~0.74 | Strongest standalone feature group |
| Elo + Recent Form | ~63.2% | ~0.69 | Limited improvement over Elo |
| Elo + Rolling Stats | ~63.7% | ~0.69 | Rolling stats provide additional signal |
| Market + Rolling Stats | ~66.9% | ~0.73 | Rolling stats complement market probabilities |
| Elo + Market Probabilities | ~67.0% | ~0.74 | Elo provides limited incremental gain over market |
| Elo + Rolling + Market | ~67.1% | ~0.73 | Strong combined baseline |
| Elo + Rolling + Market + Similar-Opponent Features | ~68.0% | ~0.76 | Best current generalization result |
| All Available Pre-Match Features | ~68.2% | ~0.77 | Current strongest configuration |

## Main Observations

- bookmaker market probabilities are currently the strongest feature group,
- Elo remains the strongest non-market feature,
- rolling performance statistics provide measurable incremental value,
- similar-opponent-strength features showed promising improvements in ROC AUC,
- Logistic Regression currently generalizes more consistently than Random Forest under temporal validation.
